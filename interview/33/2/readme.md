2. DNS劫持是是什么？（记录域名与IP的映射关系）
- 发送请求时（百度网站），去到下一个服务时被劫持了，去到了别的服务器。
1. 本地劫持：用户电脑被安装恶意软件
2. 路由器的DNS劫持：攻击者通过手段接管了路由器，从而使连接到该路由器的用户都受到影响
3. 中间DNS攻击的人：攻击者拦截用户和DNS服务器之间的通信。
4. 流氓DNS服务器：攻击者直接攻击DNS服务器
