浏览器在拿到页面所有的组件（html,css,js,图片）之后，会解析生成两个内部数据结构
--Dom树和 渲染树

1. Dom树：页面结构

2. 渲染树：表示Dom结点如何显示

3. 浏览器绘制



# 回流（重排）
    页面布局和元素几何属性的改变会导致重排
    1.改变窗口大小
    2.刷新页面（页面初次渲染）
    3.改变元素大小
    3.改变元素位置
    4.增加或删除可见的Dom元素
    重排：浏览器将渲染树中的某些结点失效又重新机选并生成新的渲染树的过程

# 重绘
   对应渲染树将结点绘制在页面上

   元素 非 几何属性的改变会导致重绘

   重排必定重绘，重绘不一定重排
